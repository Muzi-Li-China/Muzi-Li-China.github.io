# 依赖说明

提示

stylelint：运行工具
stylelint-config-standard：推荐配置
stylelint-config-prettier：防止与prettier规则冲突，将它放在extends数组的最后，可以覆盖prettier的配置
stylelint-scss：scss拓展，增加支持scss语法
stylelint-order：强制按照规定的顺序编写 css
